Advanced Mathematical Calculations & Formatting (C++)
Цей проєкт реалізує програму для розрахунку складних функцій з акцентом на чисту архітектуру коду та структуроване виведення результатів.

Технології:
C++.

Модульність: Код розбитий на функціональні блоки (обчислення та відображення), що полегшує його підтримку та тестування.

Бібліотека <cmath>: Для виконання складних операцій, таких як піднесення до степеня (pow), обчислення тригонометричних функцій та квадратного кореня.

Бібліотека <windows.h>: Для коректної роботи з кириличними шрифтами в консолі.

Що робить ця програма:
Багатофункціональні обчислення: Програма паралельно розраховує дві незалежні функції $y$ та $z$ на основі вхідних параметрів.

Виділена функція виводу: Впроваджено функцію printResults, яка відповідає за форматоване відображення вхідних даних та фінальних результатів, що демонструє розуміння принципу єдиної відповідальності (Single Responsibility Principle).

Точність даних: Використання типу double забезпечує необхідну точність для наукових та математичних розрахунків.

Інтерактивність: Програма взаємодіє з користувачем через консольний ввід, забезпечуючи гнучкість розрахунків.
